◆ sedi_uart_enable_unsol_rx()

int sedi_uart_enable_unsol_rx ( IN sedi_uart_t  uart,
IN sedi_uart_unsol_rx_t *const  unsol_rx 
)

Enable unsolicited receive.

This function enables unsolicited receive using the buffer provided. The buffered data is maintained as a circular buffer and oldest unread data is overwritten in the event of received data exceeding the buffer length. On receiving any data on the uart device , user callback is triggered indicating the status and length of the accumulated data. User may call the sedi_uart_get_unsol_data API to read the accumulated data. Caller must set the unsol_rx_callback field in the sedi_uart_unsol_rx_t otherwise error would be returned.

Parameters
[in]uartUART port index.
[in]unsol_rxpointer to unsolicited receive structure of type sedi_unsol_rx_t.
Returns
Standard return code for SEDI.
Return values
SEDI_DRIVER_OKon success, negative value for possible errors.

◆ sedi_uart_get_status()

int sedi_uart_get_status ( IN sedi_uart_t  uart,
OUT uint32_t *const  status 
)

Get UART status.

Retrieve UART interface status. Return SEDI_UART_BUSY if transmitting data; SEDI_UART_IDLE if available for transfer; SEDI_UART_TX_ERROR if an error has occurred in transmission.

The user may call this function before performing an UART transfer in order to guarantee that the UART interface is available.

Parameters
[in]uartWhich UART to read the status of.
[out]statusCurrent UART status. This must not be NULL. The returned status is to be interpreted using the sedi_uart_status_t for different error conditions.
Returns
Standard return code for SEDI.
